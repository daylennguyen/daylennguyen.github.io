# daylennguyen.github.io

Personal website built with Next.js and deployed to GitHub Pages.

## Tech Stack

- **Framework:** Next.js 16
- **Styling:** Tailwind CSS 4
- **Font:** Alagard (pixel art font)
- **Deployment:** GitHub Pages via GitHub Actions

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
# Build for production (static export)
npm run build
```

Output is generated in the `out/` directory.

## Deployment

The site automatically deploys to GitHub Pages when pushing to the `master` branch.
